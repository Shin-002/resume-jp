# 職務経歴書
Shin-002の職務経歴書です。
## 現職
[TXPMedical株式会社](https://txpmedical.jp/) にて、自治体向けプロダクト開発に正社員エンジニアとして関わっています。
### 主な業務
チーム、開発基盤の整備や新規案件に関わるシステム構築、既存アプリケーションの改修などを担当しています。

チーム、開発基盤の整備においては、オンボ資料作成、ドキュメント整備、エンジニア採用、CICD構築、AIを用いた開発環境の整備など。
開発に関しては、必要に応じてAWS（Terraform）を用いたインフラ構築からUIの実装までなんでもやります。

最近は特にAIを用いた開発環境の構築に興味があります。
取り組みとして以下のようなことを行なっています
- CIにAIコードレビューの導入
- CursorRulesを用いたガードレールの整備（e.g. 開発関連知識, 外部コンテキストインジェクション攻撃への防御）
- MCPツールの利活用（e.g. NextMCP, PlayWrightMCP, Context7）

AI関連は今後以下にチャレンジしたいと思っています。
- 必要だがパッと渡せないコンテキストをどのように設定するか（e.g. ドメイン知識）
- 運用体制の強化（e.g. エラー一次対応）
- デザイン作成におけるAIエージェントの利活用
